MangaR

MangaR is a program to read manga from the internet or your disk.

  • Download ZIP
  • Download TAR
  • View On GitHub

This project is maintained by Clepto

Welcome to MangaR's webpage

MangaR is an application for reading your favorite manga from the internet without even opening a broswer or your local collection. It supports cbr/cbz/sb7 files. You can choose through a list of 40 mangas!

Install

Ubuntu/Debian:

$ sudo apt-add-repository ppa:christriant/ppa
$ sudo apt-get update
$ sudo apt-get install mangar
